\chapter{\IfLanguageName{dutch}{Ontwikkeling van de tool}{Methodology}}%
\label{ch:ontwikkelingTool}

De gemaakte tool is ontwikkeld om een voorbereiding te zijn om MCDMA technieken te kunnen toepassen. Er is zo veel mogelijk gekozen om generiek te werken. De herbruikbaarheid van de code is altijd in het achterhoofd gehouden om van beslissing tot gebruik van datasets tot rapportering over te kunnen gaan. Omdat er met python is gewerkt zullen onderzoekers of toekomistige bachelorstudenten de tool kunnnen uitbreiden. Er is voornamelijk gekozen om zeer uitleggende te werk te gaan en elke geschreven functie duidelijke documentatie te geven. Omdat de tool een onderdeel van de analyse en rapportage zal zijn is er gekozen om een output in CSV formaat prioriteit te geven. Doordat dit een open standaard is kunnnen veel programmas waar de onderzoekers bekend mee zijn voor verdere analyse deze data kunnen interpreteren.
